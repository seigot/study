```
ポーターの5フォース分析による業界構造の理解と事例研究（Enterpriseレンタカー、NYCタクシー規制、住宅市場比較）

概要（箇条書き）

1. ポーターの5フォース分析の枠組み

5つの力

買い手の交渉力（Bargaining Power of Buyers）

売り手の交渉力（Bargaining Power of Suppliers）

新規参入の脅威（Threat of New Entrants）

代替品の脅威（Threat of Substitutes）

既存企業間の競争（Industry Rivalry）

それぞれの力が強まるほど業界の収益性は低下

2. Enterprise事例

買い手：消費者、空港レンタル客、大口顧客（ディーラー関係）

売り手：車両供給業者（ディーラー、メーカー）、人材供給

代替品：他レンタカー会社、ライドシェア、公共交通機関

重要視点：

大口購入や品質重視の買い手は交渉力が高い

供給者が前方統合（自社でレンタル事業）できる場合は交渉力が高まる

3. 買い手・売り手の交渉力の決定要因

大口取引の有無

製品やサービス品質の重要度

代替供給源や代替顧客の多さ

価格交渉で歩み寄らなければ取引を断る選択肢の有無

4. 代替品の脅威

競合が少なければ価格設定の自由度が高い

競合や代替が多いと価格競争激化 → 利益低下

5. 新規参入の脅威

高い参入障壁：規制、ブランド力、資本要件 → 既存企業の利益保護

低い参入障壁：価格引き下げや投資増で参入阻止 → 利益圧迫

重要なのは「実際の参入」ではなく「参入可能性」

6. 規制ライセンスの事例（NYCタクシー）

NYCはタクシーメダリオンを厳格に制限（1937年：11,787枚 → 現在：13,587枚）

許可制により供給量を制限し、参入障壁を形成

新規参入の脅威を抑え、既存業者の価格決定力を維持

7. 希少資源の価値（住宅市場）

人口増加の比較（1977＝100）

ヒューストン：大きく上昇

NYC：緩やかな上昇

希少資源（住宅地や規制枠）の制限が価格上昇に影響

8. 業界内競争（Industry Rivalry）

代替性が高い場合 → 価格競争で利益低下

ブランドや差別化戦略がある場合 → マークアップ率と利益が向上

9. 競争構造のスペクトラム

完全競争：代替品が完全、マークアップなし

独占的競争：多くの企業、不完全代替、ある程度のマークアップ

寡占：少数企業、戦略的相互作用

独占：代替品なし、最大マークアップ

今後の課題

参入障壁の持続的影響分析

規制や許可制が中長期的に市場効率や消費者利益に与える影響を評価

ライドシェアや新興サービスの規制適用可否を検討

資源制限と価格変動の関係予測

人口動態・都市開発計画と価格の長期予測モデル構築

希少資源の価格変動リスク管理

代替品・新規参入者への対応

技術革新（EV、カーシェア、オンデマンド交通）による競合増加への対応策

サービス多角化による依存度低減

差別化戦略の強化

ブランド力強化、付加価値サービスの開発

価格競争からの脱却と利益率向上施策

業界構造変化のモニタリング

市場の集中度、企業数、価格動向を定期的に分析

完全競争〜独占のスペクトラム上での自社ポジション評価
```
