要約

金融の基本公理：お金の時間価値、リスク回避、合理性、市場競争性が前提。

対象と役割：企業金融はプロジェクト投資判断、投資家は資産配分、金融機関は仲介と信頼構築を担う。

貨幣の時間価値：現在価値（PV）と将来価値（FV）の比較が意思決定の基礎。

リターン計算：株価変動と配当から総合的に算出。

割引率の設定：無リスク利子率＋リスクプレミアムで決定。

リスクプレミアムの考え方：高リスク資産ほど高いリターン要求。

CAPMモデル：

E(r)=rf+β(RM−rf)E(r) = r_f + \beta (R^M - r_f)E(r)=rf​+β(RM−rf​)

βは市場との感応度、RM−rfR^M - r_fRM−rf​は株式市場プレミアム。

βの解釈：0で市場リスクなし、1で市場と同等、値が大きいほど高リスク・高リターン。

1. Finance-ology（金融の公理）
今日の資産は明日の資産より価値が高い（貨幣の時間価値）

多くの資産を好み、リスクは嫌う（投資とギャンブルの違い）

リスクは様々な形で現れる（金融危機、COVID-19など）

人々は合理的で市場は競争的

2. ファイナンスの対象
焦点：企業、金融セクター、投資家

企業金融：プロジェクトの投資評価（PV・NPV、リスク調整）

投資家：資産配分（分散投資・ヘッジ）

金融セクター排除の試みは限定的成功（クラウドソーシング、フィンテック等）

3. 金融セクターの役割
投資家と投資機会をマッチング

信頼が基盤、保護が必要（FRB、SEC、FDIC）

歴史：証券法（1933, 1934）、グラス・スティーガル法

金融機関例：銀行、年金基金、保険会社、投信、ヘッジファンド、VCなど

4. 金融産業の成功要因
他業種より急速に成長

金融イノベーション（インデックスファンド、MBS、デリバティブ、CDS、ETF等）

高リスク・高リターン、レバレッジ活用

5. 貨幣の時間価値
例：今100ドル vs 1年後の100ドル（5%金利なら今の100ドルは1年後105ドルに相当）

リスクゼロでの価値換算

6. 投資リターン計算例（WMT株）
購入価格69ドル→現在104ドル、配当2ドル

年間リターン53.6%

将来価値FVと現在価値PVからリターン算出可能

7. リターンの分解
株価変動によるリターン（50.7%）

配当利回り（2.9%）

配当の考慮が重要

8. 投資意思決定例
複数の支払条件（今すぐ vs 1年後 vs 分割）を比較

割引現在価値で評価し優劣を判断

9. 事例：Snapchatの買収拒否
2013年：Facebookの30億ドル買収提案を拒否

Tencentは40億ドル評価の投資提案

10. SnapchatのIPO
2017年：17ドル/株で上場、時価総額240億ドル

11. 割引キャッシュフロー（DCF）モデル
全キャッシュフローを割引して現在価値を計算

割引率は投資の期待収益率・リスク補償を反映

12. 割引率とリスク調整
割引率が高いほど現在価値は低下

投資家は高リスク資産に低価格しか支払わない

13. 期待収益率と実現収益率の違い
実現収益率はランダムで予測困難

モデルでは期待収益率を用いる

株式は平均的に債券より高いリターン

14. 割引率設定の重要性
割引率rが高いほどPVは低い

rは無リスク利子率に基づくが、それ以上になる理由を説明する必要

15. リスク＝分散
リスクの高いキャッシュフローは分散が大きい

無リスク資産（国債、CD口座など）は確定利回り

リスク資産の例：社債、不動産、株式、人材資本、PE

16. リスクとリターンの関係
高リスク資産は高リターンを要求

歴史的リターン例：

無リスク：5%

社債（AA）：9%

不動産：10%

株式：12%

PE：20%以上

17. リスクプレミアムの決定課題
資産ごとのプレミアムをどう決定するか？

無リスク利子率の決定方法

18. CAPMによるリスク調整
投資家はリスクに対して補償が必要

式：

E(r)=rf+β(RM−rf)E(r) = r_f + \beta (R^M - r_f)E(r)=rf​+β(RM−rf​)

β：市場感応度、RM−rfR^M - r_fRM−rf​：株式プレミアム

19. CAPMのコメント
β=0：市場リスクなし、β=1：市場と同等リスク

高βは高リスク・高リターン

プロジェクトごとに固有のβ

批判：市場リスクのみを考慮

20. 例：FNMA
rf=5%r_f=5\%rf​=5%、RM=12%R^M=12\%RM=12%、β=2.04

期待収益率=19.3%、リスクプレミアム=14.3%
